如何用 hugo 搭建个人博客

步骤 1：安装 Hugo

在 code 中输入

brew install hugo

安装后验证安装

hugo version

步骤 2：建立新仓库

先进入你的 D:/software

cd /d/:software

xxx 输入用户名（用小写）

hugo new site xxx.github.io.creator

用 code 打开刚创建的新目录 xxx.github.io.creator

步骤 3：添加主题

git init

git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke

echo 'theme = "ananke"' >> config.toml

步骤 4：添加一些内容

输入hugo new posts/开博大吉.md

文件会被创建在 content\posts\开博大吉.md

title: "My First Post"
date: 2019-03-26T08:47:11+01:00
draft: true

再将 true 改成 false

步骤 5：启动 Hugo 服务器

现在，在启用草稿的情况下启动 Hugo 服务器

hugo server -D

浏览至 http：// localhost：1313 /的新站点。

步骤 6：自定义主题

config.toml 在文本编辑器中打开：

baseURL = "https://example.org/"
languageCode = "en-us"
title = "My New Hugo Site"
theme = "ananke"

将 title 以上内容替换为更个人的内容,en-us 改成 ch-Hans（中文简体）

步骤 7：建立静态页面

输入hugo，会创建一个 public 的文件夹,在创建一个.gitignore 的文件，进入输入/public/ 让 public 自成一个仓库

进入 public

git add . git commit -m 第一次部署 保存关闭

步骤 8：上传

进入 github 创建 xxx.github.io 的新仓库

git remote add xxx

git push -u origin master

步骤 9：如何查看

进入 github 仓库找到 setting 在 Github pages 中 Source 中设置 master branch,保存，点击地址即可

如何上传新内容

hugo new posts/第二篇博客.md

或在 content\posts 中新建

设置完后运行

cd public/

git add

git commit 第二次部署

hugo

git push
